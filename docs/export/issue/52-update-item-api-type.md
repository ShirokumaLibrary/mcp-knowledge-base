## Description

現在update_itemではtypeフィールドの変更が禁止されている。データ整合性を保ちながらtype変更を可能にする実装が必要。

## Content

v0.9.0のTypeORM移行により、データモデルが大きく変更されるためクローズ。新アーキテクチャで必要に応じて再検討。
