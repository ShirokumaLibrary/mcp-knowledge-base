## 🔑 重要な学習事項

### Vitestモッキングのベストプラクティス
```typescript
// ❌ 問題のあるパターン
vi.mock('./module', () => ({
  MyClass: MockClass
}));

// ✅ 正しいパターン
beforeEach(async () => {
  vi.resetModules();
  vi.doMock('./module', () => ({
    MyClass: MockClass
  }));
  const module = await import('./module');
});
```
